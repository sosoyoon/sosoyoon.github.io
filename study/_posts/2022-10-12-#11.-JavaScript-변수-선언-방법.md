JavaScript 변수 선언 방법에 대해 알아보자.

## 변수 선언
변수의 이름은 영문, 숫자, _(언더바), $(달러)로 구성된다.<br>
또한, 숫자로는 시작할 수 없으며 영문의 대소문자를 구별한다.

### var 변수  
```
var name = 'var 선언 방식'
console.log(name); //변수 선언

var name = 'var 선언 방식'
console.log(name); //변수 재선언
```
